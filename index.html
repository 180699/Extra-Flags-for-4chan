Extra Flags for 4chan.

Originally made for /int/, but we support /sp/ and /pol/ now as well. Made by /ref/ and flags contributed by various /ref/ and /int/ posters. Your flag not here? Create an issue on the github page

HOW TO INSTALL (Desktop - Chrome/Firefox/Opera)

  1. Download GreaseMonkey for Firefox, TamperMonkey for Chrome or ViolentMonkey for Opera.
  2. Install the Extra Flags script: Extra Flags for int.user.js
  3. Set your region to one of the following flags
  4. Reload the page
  5. Post from Quick Reply box (QR) (top form does not work)
